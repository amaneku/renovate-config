# @amaneku/renovate-config

[![Build Status](https://travis-ci.org/amaneku/renovate-config.svg?branch=master)](https://travis-ci.org/amaneku/renovate-config)

## Installation

```
$ npm install --save-dev @amaneku/renovate-config
```

## Usage

in `renovate.json`

```json
{
  "extends": [
    "@amaneku"
  ]
}
```
